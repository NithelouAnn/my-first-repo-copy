## Installation
INSTALLATION STEPS (GITHUB)
STEP 1: DOWNLOAD THE PROJECT
1. Open the GitHub repository in your browser
2. Click the green "Code" button
3. Select "Download ZIP"
4. Extract the ZIP file to your computer
STEP 2: OPEN THE PROJECT FOLDER
1. Open the extracted folder
2. Make sure you see files like:
   - README.md
   - package.json (for Node.js projects)
STEP 3: INSTALL REQUIRED SOFTWARE
1. Install Node.js (v18 or later)
   Download from: https://nodejs.org
2. Open Terminal / Command Prompt
3. Navigate to the project folder:
   cd path/to/project-folder
4. Verify installation:
   node --version
   npm --version
STEP 4: INSTALL PROJECT DEPENDENCIES
1. Run:
   npm install
2. Wait until installation completes
STEP 5: SET UP ENVIRONMENT VARIABLES (OPTIONAL)
1. Copy environment example file:
   cp .env.example .env
   (On Windows, manually create `.env`)
2. Open `.env` file and add values:
   PORT=3000
   API_KEY=your_api_key_here
NOTE: Do NOT upload `.env` file to GitHub
STEP 6: RUN THE PROJECT
1. Start the application:
   npm start
2. For development mode:
   npm run dev
STEP 7: OPEN THE APPLICATION
1. Open your browser
2. Go to:
   http://localhost:3000
COMMON PROBLEMS & FIXES
• Command not found
- Install Node.js and restart terminal
• Port already in use
- Change PORT value in `.env`
• Dependency errors
- Delete node_modules folder and run `npm install` again
